# College Finder ‚Äî AI-Powered Course Recommendation System
An end-to-end AI project that recommends the best Australian undergraduate courses for a student based on their academic profile.

## Problem Statement
Students planning to study in Australia face difficulty shortlisting suitable courses from thousands of available programs across universities. This system solves that by taking a student's academic profile as input and returning a ranked list of the best-matching courses using a machine learning scoring model.

## Target Users
Indian undergraduate students planning to apply to Australian universities who need personalized course recommendations based on their academic background and preferences.

## Why AI?
Manually filtering thousands of courses across entry requirements, fees, rankings, IELTS scores, intakes, and preferences is time-consuming and error-prone. A scoring model trained on real course data can instantly rank all programs by relevance to a specific student profile ‚Äî something a simple search filter cannot do.

##Scope & Limitations
- Dataset covers **Australian undergraduate programs only**
- All fees are in **AUD (Australian Dollars)**
- Recommendations are based on a **scoring model**, not guaranteed admission
- Dataset was sourced from a study abroad counselling platform and may not reflect real-time cutoff changes

##  Dataset

- **Source:** Study abroad counselling platform (real course data)
- **Size:** 3,060 rows √ó 100 columns (reduced to ~28 after cleaning)
- **Coverage:** Australian undergraduate programs across multiple universities
- **Key columns used:** University name, course name, QS ranking, IELTS required, PTE score, tuition fee, CBSE entry requirement, scholarship availability, internship availability, STEM flag, preferred intakes

---

## Data Preparation (`data_preparation.py`)
- Dropped 26 single-value columns (e.g. all rows were Australia/Undergraduate)
- Dropped 7 fully empty columns (100% missing)
- Dropped internal/business columns not relevant to students
- Filled missing values using median (for numeric) and 0/False (for boolean flags)
- Parsed `EntryRequirement` string to extract CBSE score as a numeric feature
- Parsed `DisplayIntakes` string into a list
- Renamed ambiguous columns (e.g. `backlog` ‚Üí `MaxBacklogsAllowed`)
- Converted boolean columns to integers (0 or 1)
- **Output:** `data/cleaned_data.csv`

## Exploratory Data Analysis (`eda.py`)
- Distribution of tuition fees, QS rankings, IELTS requirements
- Scholarship and internship availability breakdown
- STEM vs non-STEM course split
- Entry requirement (CBSE score) distribution
- Top universities by number of courses

## AI / ML Model (`scoring_model.py`)
**Model Type:** Content-Based Scoring Model
Each course is scored against the student's profile using weighted criteria:
| Feature | Weight |
|---|---|
| IELTS eligibility | High |
| Budget vs Tuition Fee | High |
| CBSE score vs Entry Requirement | High |
| QS Ranking | Medium |
| Backlog eligibility | Medium |
| Intake match | Medium |
| Scholarship / Internship / STEM flags | Low |

- Scores are normalized between 0 and 1
- Results are sorted by score (highest first)
- Model is saved as `scoring_model.pkl` using `joblib`
- Evaluated using ranking quality and filter accuracy metrics

## Backend (`backend.py`)

Built with **FastAPI**

| Endpoint | Method | Description |
|---|---|---|
| `/` | GET | Health check |
| `/predict` | POST | Returns ranked course recommendations |
| `/chat` | POST | AI study counsellor chatbot |
| `/explain` | POST | AI explanation for a specific course |

- Input validated using Pydantic models
- All responses returned in JSON format
- CORS enabled for frontend connection
- Uses Groq LLaMA model for AI chat and explanations

## Frontend (`frontend/`)

Built with **React + Vite**

**Features:**
- Input form: IELTS score, budget, CBSE %, backlogs, preferred intake, preferred stream
- Tick-box optional filters: STEM only, scholarship, internship, no maths, no English test, fee waiver
- Paginated results (10 per page) with match score
- Course detail page with AI-generated explanation on click
- Floating AI chatbot for study counselling
- Apply search link for each course

## How to Run Locally

### Prerequisites
- Python 3.10+
- Node.js 18+
- VS Code (recommended)

### Backend Setup
```bash
# 1. Navigate to project folder
cd collrecommend

# 2. Activate virtual environment
venv\Scripts\activate          # Windows
source venv/bin/activate       # Mac/Linux

# 3. Install dependencies
pip install -r requirements.txt

# 4. Prepare data and train model
python data_preparation.py
python scoring_model.py

# 5. Start backend
python -m uvicorn backend:app --reload
```
Backend runs at: http://127.0.0.1:8000

### Frontend Setup (new terminal)
```bash
cd collrecommend/frontend
npm install
npm run dev
```
Frontend runs at: http://localhost:5173
